Rx Writer

      This program is part of an Electronic Medical Records(EMR)  application that I have been working on for years. It is not certified by Medicare and therefore will NOT allow one to obtain bonuses. I am a retired physician and I use this personally for my volunteer work.

      Rx Writer allows one to generate standard prescriptions. It is designed to make prescribing effortless for the doctor and even allows one to completely hand write the Rx or more accurately scribble it. The idea, though, is to reduce medical errors by making things legible.

      Please wait while the program loads the medication data:

      The main screen displays the Drug List:

      One must select a drug and dosage under the various categories in order to generate an Rx. I've provided the entire list with the program. Please be aware that errors in dosing and prescribing may exist in my transcription and all of the data must be verified!!

      Double Clicking displays the Print Dialog:

      The Name Age, Signature, Number of pills (etc) and Refills must be completed to proceed. Leaving Do Not Substitute Generic is the default. One may expand the Rx with comments for the Patient or Pharmacist.

      A sorted patient list can be accessed and a new patient can be added when the Name Field is blank.

      The default is a Multi-format Rx which allow 8 medications per Rx. If the length of the Sig is too long, the program defaults to the Single format Rx. One must Add an Rx. In the Multiple format one is prompted to add another which returns to the Medication List if the response is Yes:

      If the response is No, the Rx is written:

      At this point one may type additional information into the editable fields. By right clicking, one can change the Rx to a digitally signed one or hand write on the Rx. I use a Tablet PC.

      If one chooses to write on it, the graphics are all merged and typing is not allowed:

      Print displays the available devices and double clicking the list does the job. I typically print to a pdf. A screen dump of each Rx is saved in the application/Saved/ directory. One is then prompted to enter a new Rx for the same patient or begin a new patient:

      The same functionality is applied to the Single Rx option:

      Upon selecting a New Patient or Exiting the program one is prompted to save the data which can be recalled under the Open menu.

      If a Patient Education file is associated with the Rx, it can be displayed (pdf) and printed. One can also send an email using the default client.

      One may Edit the Drug List either directly from the program or externally using the default text editor. Caution should be exercised when editing the list externally since changing the format of the text file (test.txt) can result in a failure. When one chooses to externally edit the list, a back up is generated and can be restored. The same caveat and functionality exists for externally editing the Patient List.

      These same options are duplicated on the Right Click menu over the Drug List.

      Moving Up (CTRL-U) and Moving Down (CTRL-D) are duplicated on the View Menu:

      One may search the Reference section for any Selected Medication once the drug and dosage are highlighted.

      The Rx menu allows additional options.

      One can create and edit a Favorites List and recall the Complete List as well. One can also choose to hand write the entire Rx which can be digitally Signed or Unsigned. A Search dialog allows one to quickly find a specific drug. Most of the medications are listed by their generic names.

      The Help Menu allows one to display this page and additional options.

      The Medscape or the Epocrates drug sites can be viewed. One can be directed to the Medscape Drug Interactions Calculator. One needs a free subscription to either Medscape or Epocrates to use either website! The Walmart $4.00 generic and OTC lists are provided. Using cheaper medication which has a longer track record for safety is an obvious advantage.. Various reference Medical Journal websites and Pubmed can be accessed. Most of these require a subscription to get complete information. The AAFP provides free access for older (over a year) articles. The Annals of Family Practice is free as are Pubmed and Cochrane (at least for Patient Summaries and abstracts).
